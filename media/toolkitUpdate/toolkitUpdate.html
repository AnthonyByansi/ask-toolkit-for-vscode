June 2021 update

Welcome to the updated Alexa Skills Toolkit for June 2021 (version 2.6). This update introduces a number of new features and enhancements.

What's new?

Support for APL 1.6

The ASK Toolkit now supports Alexa Presentation Language v1.6 responses. Additionally, new templates supporting APL 1.6 has been added to the toolkit, and can be found during APL response template creation. More information on APL 1.6 can be found here.

Bug fixes

  • Update toolkit webviews to be actionable in vscode 1.56v (987eb1add8363a46ab33e47f7c3e508a495342df)
  • Dependency updates to fix security CVEs (3efe8d3e9603ffb97733356fcd640e6f3984bc7d)

    • Documentation

    For more details about features, see the documents at the ask-toolkit-for-vscode Github and at the Amazon Alexa website.
